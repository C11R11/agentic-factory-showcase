## Local Development: Serve Dashboard with Python HTTP Server

To serve the dashboard locally, use Python's built-in HTTP server:

1. Open a terminal and navigate to the `dashboard` directory:
   
	 ```sh
	 cd dashboard
	 ```

2. Start the HTTP server (choose one of the following commands):

	 - For **Python 3**:
		 ```sh
		 python3 -m http.server 8000
		 ```
	 - For **Python 2**:
		 ```sh
		 python -m SimpleHTTPServer 8000
		 ```

3. Open your browser and go to [http://localhost:8000](http://localhost:8000)
